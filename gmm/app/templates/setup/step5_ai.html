Etapa 5: Inteligencia Artificial (Opcional)

Esta etapa e opcional. Se nao configurar agora, as seguintes funcionalidades nao estarao disponiveis:
  • Transcricao automatica de audios do WhatsApp
  • Abertura de OS por comando de voz
  • Chatbot inteligente para responder duvidas
Configuracao OpenAI
Obtenha em: platform.openai.com/api-keys
A OpenAI inclui o Whisper para transcricao de audio e GPT-3.5/4 para chat e NLP. Uma unica chave para tudo.
Configuracao Google Gemini
Obtenha em: aistudio.google.com/app/apikey
O Flash e mais rapido e economico. O Pro e mais preciso para tarefas complexas.
Transcricao de Audio: O Gemini 1.5 possui capacidade nativa de transcricao de audio. Voce tambem pode configurar opcoes adicionais abaixo.
Opcoes Adicionais de Transcricao (Opcional)

Use o Google Cloud STT para maior precisao em audios longos ou com ruido. E uma API separada do Gemini.

Configure em: Google Cloud Console

Use o Whisper da OpenAI como alternativa. Util se ja tiver uma chave OpenAI.

Ordem de prioridade: O sistema tentara transcrever na ordem:
  1. Gemini 1.5 (nativo, gratuito)
  2. Google Cloud STT (se configurado)
  3. OpenAI Whisper (se configurado)
